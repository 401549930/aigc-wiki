# 电商产品图提示词模板

## 三模式体系

### 模式一：白底棚拍（电商主图）
```
E-commerce product photography, [产品] on pure white background,
studio lighting. Front view, centered composition, [材质] surface
with subtle highlights. Sharp focus, commercial photography,
Amazon-ready, 1:1 square ratio, no shadow, no watermark.
```

### 模式二：场景图（详情页/社交媒体）
```
Lifestyle product photography, [产品] placed on [桌面材质] table
in a [环境] setting. [装饰元素] around the product. [光线描述]
creating soft natural shadows. [色调] color grading, editorial
catalog quality. [比例].
```

### 模式三：广告海报（投放素材）
```
Premium product advertising poster, [产品] as hero element with
dramatic [光线] lighting, [背景] with subtle texture. Brand name
"[品牌]" in [字体] typography. Tagline "[广告语]" below.
Luxury commercial photography aesthetic, [比例].
```

## 行业模板

### 香水/化妆品
```
Luxury perfume bottle product photography. [品牌] glass bottle with
[颜色] perfume liquid inside, [瓶盖描述] cap. Placed on [表面材质],
warm [光线] creating refraction effects on glass. Thin wisp of smoke
curling around the bottle. Brand label "[品牌]" in elegant [字体].
Condensation droplets on glass. Dark moody background, high-end
commercial photography, 3:4. No watermark.
```

### 运动鞋
```
Sneaker product photography, [品牌] [型号] in [配色] colorway.
Dynamic floating angle, dramatic [光线] lighting from [方向].
[鞋面材质] upper detail clearly visible, [中底] midsole profile.
[背景：纯色/渐变/场景]. Subtle motion blur suggesting energy.
Commercial footwear photography, Nike/Adidas campaign quality.
1:1 square ratio. No watermark.
```

### 护肤品套装
```
Skincare product line photography. [品牌] collection of [数量] products
arranged on [表面]. Bottles and jars of [产品1], [产品2], [产品3],
each with clean white labels. [装饰元素：花瓣/水滴/植物] styling.
Soft diffused lighting, clean white/cream background, editorial beauty
photography. Product labels readable. 16:9 hero image. No watermark.
```

### 食品包装
```
Food packaging product photography. [产品名] package design clearly showing
[包装特征], placed on [表面] with [背景]. [食物] artfully spilling from
opened package. [装饰配料] scattered around. [光线] with warm appetizing
tones, food photography styling. Package text and branding readable.
1:1 square. No watermark.
```

### 手表/首饰
```
Dramatic luxury watch advertising image. [品牌] [型号] at slight
three-quarter angle, showing [表盘描述] dial, [表带描述] strap,
brushed [金属] case reflecting ambient light. Background: dark gradient
with subtle [纹理]. Dramatic single light source from upper-left creating
[光影效果]. Luxury Swiss watchmaking aesthetic. No text except brand
logo on dial. No watermark.
```

## 材质描述速查

| 材质 | 关键词 | 典型产品 |
|------|--------|----------|
| 玻璃 | glass refraction, transparency, condensation droplets | 香水、酒瓶 |
| 金属 | brushed aluminum, metallic sheen, chrome reflection | 手表、电子产品 |
| 皮革 | leather grain, stitching detail, patina | 鞋、包、钱包 |
| 织物 | fabric texture, weave pattern, drape, wrinkle | 服装、家纺 |
| 食物 | food gloss, steam, sauce drip, crumb texture | 食品、饮料 |
| 陶瓷 | ceramic glaze, matte finish, handcrafted imperfections | 杯子、花瓶 |

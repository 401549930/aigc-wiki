# 自回归图像生成

## 定义

自回归图像生成（Autoregressive Image Generation）是 AI 图像生成的一种架构范式。与扩散模型（Diffusion Model）从噪声中逐步去噪的方式不同，自回归模型采用类似 GPT 系列语言模型的原理：将图像视为一个像素序列，逐步预测并"绘制"出每一个像素。

GPT Image 2 是这一架构最具代表性的商业产品。

## 与扩散模型的对比

| 维度 | 自回归图像生成 | 扩散模型 |
|------|---------------|----------|
| 生成原理 | 逐步预测像素序列，类似"画画" | 从噪声中逐步去噪还原图像 |
| 文字渲染 | 准确率 >95% | 较差（容易拼写错误） |
| 指令遵循 | 极强（可处理 7-8 个约束） | 一般 |
| 代表产品 | GPT Image 2, DALL-E 系列 | Stable Diffusion, Midjourney |
| 推理能力 | 生成前会先"思考"规划构图 | 直接去噪生成 |
| 迭代优化 | 对话式，可记忆上下文 | 通常需每次重新生成 |
| 开源生态 | 有限 | 极其丰富（社区模型、插件） |
| 定制自由度 | 中等 | 完全可控 |

## 历史演进

1. **GAN 时代 (2014-2020)**：生成对抗网络，如 StyleGAN、BigGAN。局限：无法用自然语言控制
2. **扩散模型时代 (2020-2025)**：CLIP 技术使文生图成为可能，催生 Stable Diffusion、Midjourney、DALL-E 3
3. **自回归时代 (2026-)**：GPT Image 2 标志性产品，先推理后生成，文字渲染和指令遵循大幅提升

## 关键技术特点

- **推理前置（Thinking）**：模型生成前先分析指令、规划构图、检查约束，然后再生成像素
- **序列化图像表示**：将图像转化为 token 序列，复用语言模型的成熟技术
- **对话式上下文**：同一对话中可基于前序图片进行迭代优化

# 角色设计提示词模板

## IP 角色还原五步法

1. **研究视觉DNA**：收集 10+ 参考图，提炼 3-5 个不可省略的视觉特征
2. **确定目标画风**：写实 / 动漫 / 水墨 / 游戏原画
3. **具体描述替代名字**：用视觉特征代替角色名，避免版权问题
4. **加入标志性道具**：每个经典角色都有标志性的道具/配件
5. **测试迭代**：生成后对比原型，调整细节

### 案例：名侦探柯南
```
❌ 名侦探柯南的角色海报
✅ A young boy detective, 7 years old appearance, round black-framed glasses,
   blue blazer over white shirt, red bowtie, signature cowlick at the front
   of his hair, holding a magnifying glass, confident investigative pose
```

## 角色设定表模板
```
Official character reference sheet for [角色名], [画风] style.
Full body front view, wearing [服装描述], [标志性道具].
[外貌特征] with [发型] and [眼睛颜色]. Multiple angle views:
front, side, 3/4 profile, back. Clean white background, professional
character sheet layout, dimensions and color codes annotated on side.
```

## 多角度转盘模板
```
Character turnaround sheet. [角色描述], [画风] style.
Five views on clean white background: front, 3/4 left, side profile,
3/4 right, back. Consistent [服装/装备] across all angles.
Color palette swatches below. Height marker on the side.
Professional character design sheet layout, animation production quality.
No watermark.
```

## 表情包设计模板
```
Character expression sheet. [角色描述] in [画风] style.
Grid of [数量] different expressions: happy, sad, angry, surprised,
confused, smug, crying, laughing, embarrassed, sleepy.
Consistent character design across all expressions, clean [背景色] background.
Chibi/SD proportion ([头身比]头身). Sticker-ready quality. No watermark.
```

# AIGC Wiki

> AIGC（AI Generated Content）图片与视频生成知识库

## 内容领域

- **文生图**: GPT Image 2, Stable Diffusion, Midjourney 等
- **提示词工程**: Prompt Formula, 风格融合, 模板库
- **商业应用**: 电商/人像/海报/UI 等场景的提示词方案
- **学习路径**: 从零基础到商业级出图的系统学习笔记

## 目录结构

```
aigc-wiki/
├── concepts/          # 核心概念与原理
├── entities/          # 工具与平台
├── learning/          # 学习笔记与日志
│   ├── logs/          # 每日学习日志
│   ├── portfolio/     # 作品集
│   └── prompts/       # 提示词模板库
├── copilot/           # Copilot 会话与自定义提示词
├── raw/               # 原始资源（文章、笔记等）
├── index.md           # 内容索引
├── log.md             # 更新日志
└── SCHEMA.md          # 知识库规范
```
